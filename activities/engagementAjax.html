1. What is AJAX?

AJAX (Asynchronous JavaScript and XML) is a technique for making requests to a server and updating web pages without reloading the whole page.

2.Why do you put your JS Code in a separate file?

Putting JS code in a separate file improves code organization, reusability, and maintainability, and helps with faster page load times due to caching.
