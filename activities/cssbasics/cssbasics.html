1.Explain the difference between inline, embedded, and external CSS.

Inline CSS applies styles directly within an HTML tag, embedded CSS is placed within a style tag in the HTML document's head, and external CSS is linked to the HTML document via a separate CSS file.

2. Explain why you should only use external CSS.

Using external CSS is beneficial because it keeps your HTML clean and organized, allows for easier maintenance by separating content from design, enables reuse of styles across multiple pages, and improves loading times since the CSS file can be cached by browsers.

3. What is a selector used for?

A selector is used in CSS to target specific HTML elements so that styles can be applied to them. It determines which elements will be affected by the defined styles.
