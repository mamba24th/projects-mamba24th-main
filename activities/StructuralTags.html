1.What goes in the header?

In HTML, the header contains metadata about the document, such as the title, links to stylesheets, and scripts, and is enclosed within the tags.

3. What goes in Main?

The main element contains the main content of the webpage, such as articles, sections, or any information that is directly related to the topic of the page. It should be unique to that page and is where the primary content is displayed.

4. What goes in the Footer?

The footer contains information at the bottom of the page, such as copyright notices, links to privacy policies, contact information, and additional navigation links. It typically provides closing details about the website.

5. HTML Validation

HTML validation is the process of checking your HTML code against web standards to ensure it is correctly structured and free of errors. This helps ensure that your webpage displays properly across different browsers and devices. You can use online tools like the W3C Validator to check your HTML for compliance with standards.
